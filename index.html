My Story

I am a full-stack developer driven by the desire to solve unique problems creatively and efficiently. Focused on constant growth and continuous education I am currently attending a 12-week career accelerator where I have the opportunity to collaborate on projects while learning and utilizing the latest tools and technologies in web development.

Before graduating Summa Cum Laude with a Bachelors degree in computer science I had the opportunity to participate in a university wide software engineering technology incubator competition where my prototype for a student government process management system was selected for continued development. I collaborated with fellow students as well as university professionals designing and building a system that would be used by students and administrators. This was a rewarding experience which facilitated my interest in developing web applications. I look forward to being an asset to a company where I can continue to learn and grow as a developer.

Projects

Traxx

Traxx is a discographic music searching and album collection client application. Users can search from a database containing over 10M records from over 5M artists. It is a single page application (SPA) built with React that interfaces with an Express/Node backend API using axios for http requests.

Nozama!

Nozama is an e-commerce site for selling all kinds of products inspired by Amazon. It utilizes the Stripe API for payment authorization. The back end is an express API hosted on Heroku. The front end is a vanilla JS client.

Strikeout

Strikeout is an MLB themed tic-tac-toe game. It is a single page application (SPA) written with vanilla JavaScript that interfaces with a state managing backend API using AJAX.

Wine Collector

This is a wine tasting client application. Users can select from a large list of wines and can create tasting notes for each wine. It is a single page application (SPA) that interfaces with a Ruby on Rails backend API using AJAX.
