Traxx

Traxx is a discographic music searching and album collection client application. Users can search from a database containing over 10M records from over 5M artists. It is a single page application (SPA) built with React that interfaces with an Express/Node backend API using axios for http requests.

Nozama!

Nozama is an e-commerce site for selling all kinds of products inspired by Amazon. It utilizes the Stripe API for payment authorization. The back end is an express API hosted on Heroku. The front end is a vanilla JS client.

Strikeout

Strikeout is an MLB themed tic-tac-toe game. It is a single page application (SPA) written with vanilla JavaScript that interfaces with a state managing backend API using AJAX.

Wine Collector

This is a wine tasting client application. Users can select from a large list of wines and can create tasting notes for each wine. It is a single page application (SPA) that interfaces with a Ruby on Rails backend API using AJAX.
